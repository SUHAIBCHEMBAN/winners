# Fest Result App

A React-based web application for publishing and viewing Arts Fest results in real-time.

## Features

- **Public View:**
  - Live Scoreboard (Team Badr vs Team Uhud)
  - Real-time result listing with "On-Stage" and "Off-Stage" filtering
  - Search by name, program, or category (UP, HS, HSS, etc.)
  - Clean Professional Light Theme

- **Admin Panel:**
  - Secure login
  - Publish results with Position (1st, 2nd, 3rd)
  - Edit/Delete results
  - Dashboard statistics

## Data Structure

- **Teams:** Team Badr & Team Uhud
- **Categories:** UP, HS, HSS, JUNIOR, SENIOR
- **Programs:** Quran Recitation, Mappilapattu, Speech, etc.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run locally:**
    ```bash
    npm run dev
    ```

3.  **Build:**
    ```bash
    npm run build
    ```
